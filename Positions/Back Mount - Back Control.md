# Back Mount / Back Control

**Category:** Dominant Position (Top)

## How to Identify

One person is BEHIND the other, chest-to-back. Usually has HOOKS IN (feet inside the thighs) or a body triangle (legs wrapped around the waist). The person in front cannot see the person behind them.
